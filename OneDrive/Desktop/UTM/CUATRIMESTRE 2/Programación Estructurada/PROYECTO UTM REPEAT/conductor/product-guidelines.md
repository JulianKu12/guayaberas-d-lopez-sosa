# Product Guidelines: Premium Real Estate

## Prose Style: Elegant & Sophisticated
- **Tone:** Professional, trustworthy, and modern.
- **Voice:** Direct but approachable, using clear language that reflects the luxury nature of the properties.
- **Copy Goals:** Inspire confidence and help users navigate high-end real estate with ease.

## UX Principles: Focus on Simplicity
- **Simplicity First:** Prioritize a clutter-free interface. Focus on one major action at a time to ensure the user is not overwhelmed.
- **Visual Hierarchy:** Use typography and spacing to create a clear visual path, guiding the user to the most important property details.
- **Instant Feedback:** Ensure all search and filter actions provide immediate visual confirmation.

## Branding & Visual Identity: Earthy & Premium Greens
- **Primary Colors:** Use the established palette including "Nordic Dark" (#19322F) and "Mosque" (#006655).
- **Secondary Colors:** Support the primary greens with "Hint Green" (#D9ECC8) and clean backgrounds like "Clear Day" (#EEF6F6).
- **Typography:** Use "Inter" (sans-serif) for its clean and readable interface qualities.
- **Imagery:** Use high-quality, professional photography for all listings, ensuring they are the focal point of the discovery process.
